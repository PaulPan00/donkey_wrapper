# Image-Editor
An small and easy-to-use image editor GUI using PyQt5 and Cv2

## Usage
`pip install opencv-python`, `pip install PyQt5`, `pip install numpy`, `pip install qimage2ndarray`, `pip install scipy`

Run main.py

## Current editing options
- Crop image
- Rotate image
- Change brightness
- Change contrast
- Change saturation
- Auto contrast
- Auto sharpen
- Auto cartoon
- Auto invert
- Color mask
